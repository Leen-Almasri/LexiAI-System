# LexiAI

LexiAI is an AI-powered interactive glossary system that enhances AI terminology understanding by combining definitions from a structured glossary with real-world examples generated using Large Language Models (LLMs).

## Project Idea
The system transforms static AI glossary definitions into an interactive learning experience by:
- Retrieving definitions from an AI glossary dataset
- Generating real-world examples using LLMs
- Providing a user-friendly web interface

## Workflow
User → Streamlit UI → FastAPI → Glossary Data → LLM → Response

## Features
- AI terminology lookup
- Real-world example generation
- Interactive web interface
